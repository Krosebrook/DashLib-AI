**The Enterprise-Grade Dashboard Synthesis Engine.**

DashLib AI is an intelligent architectural tool designed for Senior Frontend Engineers. It leverages **Google Gemini 3 Pro** and the **Gemini Live API** to synthesize, audit, and deploy production-ready dashboard code with zero boilerplate.

---

## 🚀 New in v3.8: Multimodal Synthesis & Portable Apps

### 🎨 **Multimodal Synthesis Engine**
- **Vision-to-UI**: Upload a screenshot or wireframe, and DashLib will replicate the visual hierarchy, layout, and metrics using React + Tailwind.
- **Voice-to-Dashboard**: Use the **Gemini Live API** to dictate complex requirements. Real-time transcription and synthesis allow for hands-free dashboard architecting.
- **Data Context Injection**: Paste SQL schemas or Figma JSON nodes directly into the generator to map UI components to real data structures.

### 🧪 **Interactive Workbenches**
- **Security & Compliance**: Define custom alert rules (e.g., "Failed Login > 5") and click **"Simulate Traffic"** to test your thresholds against randomized telemetry data in real-time.
- **Model A/B Testing**: Select multiple models (GPT-4o, Gemini 3, Llama 3), run parallel prompt tests, and compare latency/cost visualizations side-by-side.

### 📦 **Portable Micro-Apps (PMA) v2.0**
- **React 19 + esm.sh**: Generated apps now leverage React 19 and resolve dependencies via `esm.sh` import maps for maximum performance and compatibility.
- **Zero-Install**: Download any dashboard as a **Single-File HTML** app that runs anywhere without build steps, `node_modules`, or local servers.
